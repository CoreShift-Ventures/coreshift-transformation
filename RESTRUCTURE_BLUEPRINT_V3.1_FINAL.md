# Landing Page Blueprint V3.1 FINAL (Process-First + Category Tabs + AI Layer)

## Design Philosophy - Locked In ✅
- **Process-first, not industry-first** - Universal pain points everyone recognizes
- **Product positioning** - Templates you customize, not bespoke consulting
- **Category tabs** - 3 organized groups (Grow Revenue, Run Operations, Manage Money)
- **AI as cross-cutting layer** - Embedded + dedicated section
- **screen.studio aesthetic** - Clean, visual, screenshot-forward detail modals
- **Preserve all UX/animations** - Maintain existing quality and polish

---

## Strategic Positioning

### Primary Message:
```
Transform Your Business Processes With AI-Powered Apps
Built in Weeks, Not Months
```

### Value Proposition:
- **What:** 8 proven process templates organized in 3 categories
- **How:** Customized to your business in 2-week Blueprint Sprint
- **Differentiator:** Built-in AI intelligence (ShiftIQ automation + CompassAI insights)
- **Proof:** See working prototype before you commit

### Target Audience:
- ANY fast-growing SMB (not locked to specific industries)
- Businesses with manual/scattered processes
- Teams wanting automation without complexity
- Leaders seeking AI transformation without hype

## Technical Implementation Notes

### Category Tabs:
```typescript
// State management
const [activeTab, setActiveTab] = useState<'revenue' | 'operations' | 'money'>('revenue');

// Tab content mapping
const tabContent = {
  revenue: [leadConversion, customerSuccess, revenueOps],
  operations: [orderMgmt, inventory, appointments, projects],
  money: [invoicing, expenses]
};

// Smooth transition
<AnimatePresence mode="wait">
  <motion.div
    key={activeTab}
    initial={{ opacity: 0, y: 20 }}
    animate={{ opacity: 1, y: 0 }}
    exit={{ opacity: 0, y: -20 }}
    transition={{ duration: 0.3 }}
  >
    {tabContent[activeTab].map(process => <ProcessCard {...process} />)}
  </motion.div>
</AnimatePresence>
```

### Screenshot Carousel:
```typescript
// Auto-scroll logic
useEffect(() => {
  const interval = setInterval(() => {
    setCurrentIndex((prev) => (prev + 1) % screenshots.length);
  }, 3000); // 3 seconds per image

  return () => clearInterval(interval);
}, [isPaused]); // Pause on hover

// Pause on hover
<div
  onMouseEnter={() => setIsPaused(true)}
  onMouseLeave={() => setIsPaused(false)}
>
  {/* Carousel content */}
</div>
```

### Modal Accessibility:
- Focus trap (can't tab outside modal)
- ESC key to close
- Click overlay to close
- ARIA labels
- Scroll lock on body when modal open
